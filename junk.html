Beaudryland =========== by Matt Beaudry mattbeaudry.com A simple javascript game engine inspired by Minecraft. Features 2015 - Random map terrain generation for 6 different map types - Player movement and collision detection - User register/login and a Map Gallery - Collectable resources and craftable items - Bunch of items and blocks: weapons, tools, treasures, blocks, transportation, mechanisms, musical instruments - Enemies & animals with basic AI - Projectiles - Saving and loading a player & game with php/MySQL - On screen control pad for touch devices - Basic animation for player item swing, transportation items, projectiles and animated map
